## О проекте

Сценарий поможет автоматически настроить сервер для CS-Cart и Multi-Vendor 4.0+. Протестировано в [Айхоре](http://ihor.ru/), [FastVPS](http://fastvps.ru) и [Digital Ocean](https://digitalocean.com).
